# CHANGELOG

Все важные дополнения и изменения в проекте фиксируются в этом файле.

## Формат ведения
- Записи ведутся по дате в формате `YYYY-MM-DD`.
- Для каждого дня создается отдельный заголовок `## YYYY-MM-DD`.
- Под заголовком дня перечисляются изменения, сделанные именно в этот день.
